# Barely a Source Package Manager
A source package manager written in Lua for Arch-based Linux Distributions. This is just the concept build, everything is subject to change.

This is just the first concept build. Nothing special. I created this repo to have the job of creating the repo done as well as share my progress.

## Should I use this?
Probably not. Not now, at least. It's barely tested and this is just a concept that is extremely opinionated (which hopefully will change).

## How do I install it?
I advice against using this package manager as it is right now.

But if you really wanna use it:

1. Just download the bspm.lua file
2. Run "sudo/doas cp bspm.lua /bin" (assuming you're in the same directory as the downloaded file

That's it. You have now installed it.
